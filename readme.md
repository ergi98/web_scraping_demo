# Web scraping demo

A simple python script demonstrating how to web scrape different information form websites using `lxml`, `bs4` and `requests`.

Sites being used here are made for scraping:

1. http://books.toscrape.com
2. https://quotes.toscrape.com

Information being collected:

1. A book thumbnail image scraped and saved locally with the correct content type
2. A list of all the books on [books.toscrape](http://books.toscrape.com)
3. A list of all quotes, unique authors and top 10 tags from [quotes.toscrape](https://quotes.toscrape.com)
